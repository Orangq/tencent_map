# tencent_map

### 简介

该demo采用element+vue2+JavaScriptV2 API，其中包括地图选点、标记获取经纬度，设置中心点，地点推荐词和地图缩放功能，是一个弹窗。参考文档https://lbs.qq.com/webApi/javascriptV2/jsGuide/jsOverview

### 安装依赖

```
npm install
```

### 启动项目
```
npm run serve
```
